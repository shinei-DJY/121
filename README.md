# Agent Tool 1 - 多API工具集成

这是一个基于 LangChain 的多 API 工具集成项目，包含三个实用工具：今日老婆、Steam 喜加一和今日油价。

## 功能特性

- **今日老婆**：获取二次元老婆图片，支持传入ID固定返回图片
- **Steam 喜加一**：获取最新的 Steam 免费游戏信息
- **今日油价**：获取指定省份的今日油价信息
- **大模型集成**：使用 deepseek-v3 模型智能选择和调用工具
- **工具切换**：支持灵活选择和切换不同工具

## 安装依赖

```bash
pip install langchain langchain-openai requests pydantic
```

## 快速开始

### 终端界面操作

```bash
python agent_tool_1.py
```

这将启动一个交互式终端界面，你可以通过数字选择使用不同的工具：

```
=== Agent Tool 1 - 多API工具集成 ===
欢迎使用多API工具集成系统
-----------------------------------
可用工具:
1. 今日老婆 - 获取二次元老婆图片
2. Steam喜加一 - 获取最新的Steam免费游戏
3. 今日油价 - 获取指定省份的油价
4. 智能问答 - 让AI自动选择合适的工具
0. 退出系统
-----------------------------------
请选择工具编号: 
```

### 操作流程

1. **选择工具**：输入对应工具的编号
2. **输入参数**：根据提示输入必要的参数
3. **查看结果**：系统会调用相应的API并显示结果
4. **继续使用**：完成后会返回主菜单，可选择其他工具
5. **退出系统**：输入0退出

### 示例操作

**今日老婆**：
- 输入1 → 输入ID（可选）→ 查看结果

**Steam喜加一**：
- 输入2 → 直接查看结果

**今日油价**：
- 输入3 → 输入省份名称 → 查看结果

**智能问答**：
- 输入4 → 输入问题 → 让AI自动选择工具并回答

## 工具详细说明

### 1. 今日老婆

**功能**：获取二次元老婆图片

**参数**：
- `id`（可选）：传入ID可在同一天固定返回同一张图片

**调用示例**：
- 随机获取：`给我一张今日老婆图片`
- 固定获取：`给我一张ID为10001的今日老婆图片`

### 2. Steam 喜加一

**功能**：获取最新的 Steam 免费游戏信息

**参数**：无

**调用示例**：
- `最新的Steam喜加一游戏是什么`
- `现在有什么免费的Steam游戏`

### 3. 今日油价

**功能**：获取指定省份的今日油价

**参数**：
- `province`（必填）：省份名称，如"北京"、"上海"、"广东"

**调用示例**：
- `北京今天的油价是多少`
- `上海的92号汽油价格`

## 工具切换方法

### 方法1：终端界面选择（推荐）

启动程序后，在终端界面直接选择工具编号：

```
请选择工具编号: 1  # 选择今日老婆
请选择工具编号: 2  # 选择Steam喜加一  
请选择工具编号: 3  # 选择今日油价
请选择工具编号: 4  # 智能问答模式
```

### 方法2：通过自然语言指令切换

在智能问答模式（编号4）中，直接在输入中指定要使用的工具：

```
请输入您的问题: 使用今日老婆工具获取一张图片
请输入您的问题: 使用Steam喜加一工具查询最新免费游戏
请输入您的问题: 使用今日油价工具查询上海的油价
```

### 方法3：通过问题内容自动切换

在智能问答模式中，大模型会根据问题内容自动选择合适的工具：

```
请输入您的问题: 我想要一张二次元图片  # 自动选择今日老婆
请输入您的问题: Steam最近有什么免费游戏  # 自动选择Steam喜加一
请输入您的问题: 深圳的油价  # 自动选择今日油价
```
